# Quant_Project_2 - Macro Regime Strategy

This project investigates whether macro regimes derived from cross-asset financial signals can predict systematic differences in equity options implied volatility surfaces. The framework combines data collection, HMM-based regime classification, options surface analysis, and a cost-adjusted backtest.

Built as part of self-directed exploration into quantitative signal research.
